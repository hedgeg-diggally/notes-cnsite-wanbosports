# notes-cnsite-wanbosports

## 项目简介

`notes-cnsite-wanbosports` 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面以静态文件形式存放，不针对任何特定域名或网站，仅作为资料整理与展示用途。

## 目录说明

```
notes-cnsite-wanbosports/
├── README.md          # 本说明文件
├── pages/             # 存放独立 HTML 页面文件的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 页面引用的资源文件（如 CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下的每个 `.html` 文件均为一个独立页面，可直接通过浏览器打开。
- `assets/` 目录用于存放页面所需的公共或私有资源，保持页面结构整洁。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面可以包含文字、图片、样式等静态内容。页面之间不依赖特定后端服务，所有内容均以纯前端方式呈现。

这些页面通常用于记录、展示或测试，不涉及任何实时交互或动态数据请求。如果您需要查看某个页面，只需在本地或托管环境中打开对应的 HTML 文件即可。

## 维护说明

- **新增页面**：在 `pages/` 目录下创建新的 `.html` 文件，并确保引用的资源路径正确。
- **资源管理**：如需添加图片、样式或脚本，请统一放置在 `assets/` 对应子目录中，并在页面内使用相对路径引用。
- **更新记录**：建议在每次更新时简要记录变更内容，以便追踪历史版本。
- **兼容性**：页面尽量使用基础的 HTML/CSS/JavaScript 语法，以保持较好的浏览器兼容性。
